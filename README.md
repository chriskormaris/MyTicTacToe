# MyTicTacToe
The classic TicTacToe game with a nice GUI, made with Java Swing.

Game modes:
1) Human vs Minimax AI
2) Human vs Human
3) AI vs AI
4) Client-Server (like Human vs Human for different PCs, using static IPs)
